# 🍽️ Restaurant Order Management System 

## 📌 Overview

This project is a **console-based Restaurant Order Management System** developed using C++. It allows users to view a menu, select items using unique IDs, enter quantities, and generate a final bill.

The project demonstrates the use of fundamental programming concepts to build a simple real-world application.

---

## 🚀 Features

* 📋 Display of restaurant menu
* 🔢 Item selection using unique ID
* 🔁 Multiple item ordering
* 🧮 Automatic bill calculation
* ✅ Input validation for ID and quantity
* 🧾 Final bill with total amount

---
